Você é um arquiteto de software sênior especializado em aplicações educacionais escaláveis e leves para mercados emergentes.
Vamos construir uma plataforma chamada MINSA Prep.
Objetivo do sistema:
Plataforma web de simulados e treino inteligente para concursos da saúde em Angola, começando com duas áreas:
Farmácia
Enfermagem
Stack obrigatória:
React (Vite ou Next.js leve)
Supabase (Auth + PostgreSQL)
Hospedagem Vercel
API Gemini (apenas no admin para gerar questões)
Interface extremamente leve e otimizada para internet 3G
Código modular, limpo e escalável
⚠️ O sistema deve ser simples, rápido, performático e fácil de manter por um único desenvolvedor.
